# hirain-metamodel-table


## 项目说明
本项目是基于[vxe-table](https://www.npmjs.com/package/vxe-table)@4.11.22修改的版本，感谢原作者[x-extends](https://www.npmjs.com/~x-extends)的贡献


## 修改内容
 - 增加了`insertTreeData`函数，实现更高性能的树结构数据插入功能
 - 增加了`removeTreeData`函数，实现更高性能的树结构数据删除功能
 - 增加了`replaceTreeData`函数，实现更高性能的树结构同级节点顺序修改
 - 增加了`exchangeTreeData`函数，实现更高性能的树结构同级节点位置交换
